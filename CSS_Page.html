CSS

CSS stands for Cascading Style Sheets.
CSS describes how HTML elements are to be displayed on screen, paper, or in other media, and can control the layout of multiple web pages all at once.
While html creates the actual content on the page, css is responsible for the design or style of the website, including the layout, visual effects and background color.
There are two ways you can link css and html class.
You can either put it inline through the use of the style attribute or use it externally through the use of a link element that connects a stylesheet to the HTML.
The CSS syntax includes the selector, property, and value.
  • The selector decides what group is going to be changed.
  • The property decides how that group is going to be changed.
  • The value decides how much of the property of the group is going to be changed.
By putting the ; sign, you can have mutiple changes done to your group.
These are some of the things css can do.
