CSS stands for Cascading Style Sheets.
CSS describes how HTML elements are to be displayed on screen, paper, or in other media, and can control the layout of multiple web pages all at once.




While html creates the actual content on the page, css is responsible for the design or style of the website, including the layout, visual effects and background color.
There are two ways you can link css and html class.
You can either put it inline through the use of the style attribute or use it externally through the use of a link element that connects a stylesheet to the HTML.





Syntax

The CSS syntax includes the selector, property, and value.

    The selector decides what group is going to be changed.

    The property decides how that group is going to be changed.

    The value decides how much of the property of the group is going to be changed.

By putting the ; sign, you can have mutiple changes done to your group.


These are some of the things CSS can do.

Using CSS, you can change the color of texts by using the formula,
selector {
color: color
You can also change the background color and border color by putting background-color or border-color in place of color and choosing the color you want.

To add or change the font size and style, you can use the properties, font-size and font-family.
The value of font-size is determined by numbers and the value of font-family is the name of the font style, so for example, Times New Roman.

To align texts you use the property, text-align, then put a value.
Some examples of the values are center, left, and right.




Classes vs IDs

Classes and IDs are both used to make applying CSS easier.
However, classes are used when the same style is used on multiple elements, while IDs are used when a style is unique to a single element.
After assigning a class or id on a element, you can choose that element in CSS by using a period for a class, and a hashtag for an id.


CSS Box Model


The box model includes the content, padding, border, and margin.

    Content: The inner content of your HTML element. This is the area where text would be displayed.

    Padding: A box that surrounds an HTML element's inner content. It is used to expand the space between the content and the border.

    Border: A box that surrounds an HTML element's padding.

    Margin: A functionally invisible box that surrounds an HTML element's border. The margin is meant to act as white space to separate HTML elements from each other.


CSS Display Properties

In CSS, there are block properties and inline properties.
Blocks always start on a new line and take up the full width, stretching out to the left and right as far as it can.
On the other hand, an inline element starts wherever you put it and takes up only as much space as necessary.

    section

Inline

    strong

Floating


The Float property is used to position and format content.
It lets you place an element on the left or right side of its container, allowing text and inline elements to wrap around it.
Floating can be done by sing the formula,
div {
float: value
}
The value can be left or right to push it to side.

Position Properties

There are 5 position properties in CSS, and they are static, relative, fixed, sticky, and absolute.
Static is the default position when applied.
Relative positions the element relative to the normal position it would otherwise have to offset an element from its default.
Fixed will make it always stay where you put it even if a user scrolls down the page.
Sticky is positioned based on the user's scroll position. It appears fixed until a user tries to scroll, then moves along with the user, overriding other elements.
Absolute elements are placed in absolute position based either on their parent container or the overall document body.
