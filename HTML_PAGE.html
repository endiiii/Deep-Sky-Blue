HTML 5



Terms

  • HTML: Hyper Text Markup Language
  • Document Declaration: Declares what language you are using
  • Head tag: This where you can put stuff like css, titles, and other things that are not directly seen on the page
  • Heading tag: Note that this is different from the head tag. It goes from h1 to h6, and the font size decreases as you go higher up in numbers.
  • Paragraph tag: Like the name suggests, you can put paragraphs within those tags
  • Lists: There are two types of lists. Ordered lists have numbers while unordered lists have bullet points.

    • Logic

  • The website starts with a html tag and ends with a closing html tag
  • The head tag goes first, then the body tag
  • The bulk of what you see goes within the body tag
  • The heading tags are good for organizing the paragraphs by giving them a common name to go under

    • Bonus

  • You can use HTML with other languages as well. Javascript, jquery, and css (if you count that as a programming language) can work nicely with it.
  • You can inspect and debug using chrome by using the inspect tool.
  • Using text editors like Visual Studio Code can help a lot with coding
  • It is often convenient to use w3schools or other websites in case you forgot things like syntax.
